⚠️ Pain Points
▲MarianaTek API latency during email validation can feel like rejection
▲No SSO — manual password creation adds friction
▲Notification permission prompt before showing app value
▲12-page slider is long — no skip option, no progress save
▲11 studio layouts for spot preference = cognitive overload
▲Height/weight unit toggle can confuse
▲Hero card state machine is invisible to users
▲Filter reset on back-navigation loses selections
▲Spot selection per class in multi-booking is repetitive
▲3-phase weekly check-in feels marathon-like
▲Fri 9AM–Sun midnight window competes with weekends
▲"Remind me later" exits lose partial progress
▲AI transparency: can't see why KYN recommends swaps
▲Goal alignment score without benchmarking feels abstract
▲Multiple API calls create loading states during swaps
▲Dream instructor: 5 bipolar sliders feel abstract
▲Delete account is irreversible with no cool-off
▲No way to export or share fitness data
