# S&T User Journey Map — Deploy to Vercel

## One-command deploy

```bash
cd deploy-to-vercel
npx vercel --prod
```

That's it. Follow the prompts (select your Vercel account, confirm project name).

## What this is

Interactive user journey map for the Sweat & Tonic member app covering:
- 6 lifecycle phases (Activate → Onboard → Engage → Reflect → Optimize → Retain)
- 9 major flows, 50+ screens
- Clickable touchpoints with drill-down details
- Pain points, opportunities, emotional arc, API layer
- Persona filtering (New Member / Active / Power User)
